## 📦 Как применить новый дизайн

### Шаг 1: Создайте бэкап (обязательно!)

```bash
cd landing
mkdir backup
copy index.html backup\
copy styles.css backup\
copy script.js backup\
```

### Шаг 2: Замените основные файлы

```bash
# Windows
copy index-new.html index.html /Y
copy styles-new.css styles.css /Y
copy script-new.js script.js /Y

# Linux/Mac
cp index-new.html index.html
cp styles-new.css styles.css
cp script-new.js script.js
```

### Шаг 3: Проверьте сайт

```bash
# Откройте index.html в браузере
start index.html
```

### Шаг 4: Если что-то пошло не так

```bash
# Восстановите из бэкапа
copy backup\index.html index.html /Y
copy backup\styles.css styles.css /Y
copy backup\script.js script.js /Y
```

## 🎨 Кастомизация

### Изменить основной цвет:

Откройте `styles-new.css` и найдите:

```css
:root {
    --primary: #6366f1;        /* Основной цвет */
    --primary-dark: #4f46e5;   /* Темный */
    --primary-light: #eef2ff;  /* Светлый */
}
```

Замените на свои цвета, например:
```css
:root {
    --primary: #ef4444;        /* Красный */
    --primary-dark: #dc2626;
    --primary-light: #fef2f2;
}
```

### Изменить города:

В `index-new.html` найдите:
- "Хабаровск" → замените на ваш город
- В provider-card измените районы
- В отзывах измените города

### Добавить реальные данные:

Вместо демо-данных вставьте реальные:
- Отзывы → замените на настоящие
- Рейтинги → реальные оценки
- Цены → актуальные
- Сервисы → реальные компании
