
### Вариант 2: Применить новый дизайн

```bash

cd landing
mkdir backup
copy index.html backup/
copy styles.css backup/
copy script.js backup/

# 2. Замените старые файлы на новые
copy index-new.html index.html
copy styles-new.css styles.css
copy script-new.js script.js

# 3. Удалите временные файлы
del index-new.html
del styles-new.css
del script-new.js
```
